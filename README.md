# RoofCode NZ

A Vercel-ready React app for asking NZ roofing guidance questions.

## Deploy
1. Upload all files in this folder to a new GitHub repo.
2. Import the repo into Vercel.
3. Add either `ANTHROPIC_API_KEY` or `OPENAI_API_KEY` in Vercel Environment Variables.
4. Deploy.

The app works best with an API key. If no key is set, it still returns a local starter guidance response.
